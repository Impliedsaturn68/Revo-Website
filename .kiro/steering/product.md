# Product Overview

Browser Start Page is a lightweight, productivity-focused dashboard application designed to serve as a browser home page. It provides essential tools for time management and quick navigation without external dependencies.

## Core Features

- Time-based greeting display with live clock
- 25-minute focus timer for productivity sessions
- Task list for managing to-do items
- Quick links for favorite website shortcuts
- Client-side data persistence using Local Storage

## Design Philosophy

- Simplicity: No external dependencies, vanilla JavaScript only
- Performance: Fast load times (<500ms), responsive interactions (<100ms)
- Persistence: Reliable data storage across browser sessions
- Compatibility: Works across Chrome, Firefox, Edge, and Safari (modern versions)
